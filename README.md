# Collectez-des-donnees-en-respectant-les-normes-RGPD
Ce projet est composé d'une activité principale. 

Vous réaliserez une mission : aider Dev’Immediat, une filiale de l'assurance automobile, à traiter les dossiers de demande d'assurance automobile reçus et réaliser votre première collecte de données en respectant le Règlement Général de Protection des Données.
